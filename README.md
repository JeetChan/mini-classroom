# mini-classroom

轻量级小学知识点交互学习网站集合，涵盖数学、语文、英语多学科基础内容。专为家长、授课老师及小学生打造，助力课后辅导与趣味启蒙。

---

## 项目理念

「小教室」不是另一个刷题平台。每个站点都围绕**一个可聚焦的知识体系**构建——不是堆砌内容，而是精选、编排、交互化。让孩子和家长可以打开即学，不需要注册、登录、下载。

四条设计原则：

- **零门槛**：纯静态 HTML，双击即用，不依赖数据库或服务端
- **可离线**：核心站点支持 PWA，安装后可离线使用
- **自驱学习**：闪卡、进度追踪、内联测试，把学习反馈做到即时
- **家长友好**：知识体系透明，数据结构清晰，方便定制

## 各站点详细介绍

### math-grade1-2-workshop — 一年级下册数学

基于人教版一年级下册数学教材，23 个知识点覆盖五大章节：

| 章节 | 内容 |
|------|------|
| 认识图形（二） | 长方形、正方形、三角形、圆、平行四边形、七巧板 |
| 20 以内退位减法 | 破十法、平十法、想加法算减法 |
| 分类与整理 | 单一标准分类、多标准分类、统计表 |
| 100 以内数的认识 | 数数方法、数的组成、读写规则、百数表、比大小 |
| 认识人民币 | 人民币单位、换算、计算、付钱方法 |

功能：渐进式学习 → 闪卡复习 → 知识点索引 → 进度追踪 → 详情页（含例题 + 四选一测试）。

[→ 站点 README](./math-grade1-2-workshop/README.md)

### chinese-grade1-site — 一年级下册语文

配套人教版一年级下册语文教材，六大模块：

| 模块 | 内容 |
|------|------|
| 古诗文 | 7 首必背古诗（春晓 / 静夜思 / 赠汪伦 等），含诗意解析、意象分析、修辞说明 |
| 课文背诵 | 10 篇课文道理速记、中心思想、背诵卡片 |
| 日积月累 | 歇后语、天气谚语、学习名言、家庭四字词、对韵成语 |
| 词语积累 | AABB / ABB / ABAB 叠词、拟声词、四季词语 |
| 近反义词 | Tab 切换近义词 / 反义词，悬停查看例句 |
| 句式仿写 | 8 种句式（比喻句、「一边…一边…」「要是…就…」等） |

另有闪卡复习功能，5 个卡组可键盘操作翻卡。

[→ 站点 README](./chinese-grade1-site/README.md)

### basic-english-workshop — Basic English 核心词工坊

基于语言学家 Ogden 1930 年提出的 Basic English 体系——用 **850 个核心词**覆盖日常表达 90% 的概念。本项目将其拆解为 30 个学习单元：

- 核心框架（4 个）：操作词系统、18 个核心动词、21 个方向介词、代词系统
- 分类体系（4 个）：事物词、性质词、反义词系统、国际词汇
- 主题分类（16 个）：颜色、身体、食物、家庭、动物、时间 等
- 学习方法（6 个）：-er 造词法、复合词构建、设计哲学、学习优先级

功能：渐进学习（含内联测试）→ 闪卡复习 → 词汇索引 → 进度追踪 → 详情页。

[→ 站点 README](./basic-english-workshop/README.md)

---

### basic-english-site — Ogden Basic 850

Basic English 的早期版本，更聚焦词汇本身：

- **词汇表**：850 词按 5 个类别浏览（操作词、通用名词、图示词、性质词、反义对）
- **核心动词**：18 个操作词变位表 + 与 20 个方向词的组合对照
- **造句练习**：6 种句型模板，练习「操作词 + 方向词替代复杂动词」
- **闪卡 + 进度**：翻转闪卡 + 本地进度记录

如果你想要更完整的框架讲解，推荐使用 basic-english-workshop。

[→ 站点 README](./basic-english-site/README.md)

## 快速开始

所有站点都是纯静态文件，无需安装、无需构建。

### 直接打开

```bash
# 进入任一站点目录，直接用浏览器打开首页
open math-grade1-2-workshop/index.html
```

### 本地服务器（推荐，启用 PWA）

```bash
# 进入站点目录后
npx serve .
# 或
python -m http.server 8080
```

浏览器访问 `http://localhost:3000`（serve）或 `http://localhost:8080`（Python）。

### 部署

任意静态托管平台均可：

- **Vercel**：`npx vercel --prod`
- **Netlify**：拖拽文件夹上传，或 `npx netlify deploy --prod --dir .`
- **GitHub Pages**：仓库 Settings → Pages → 选择分支和目录

## 技术栈

所有站点共享同一技术理念：

- **纯静态**：HTML5 + CSS3 + 原生 JavaScript（ES6+），零框架依赖
- **PWA**：math-grade1-2-workshop 和 basic-english-workshop 支持安装到桌面、离线使用
- **本地持久化**：学习进度通过 `localStorage` 保存，无需登录
- **响应式**：适配桌面和移动端
- **字体**：Inter（英文站点）/ Noto Sans SC（中文站点），或系统字体回退

## 自定义内容

每个站点的数据层与展示层分离，修改内容无需改 HTML：

- **数学 / 基础英语工坊**：编辑 `js/wordData.js`（知识点数据）和 `js/siteConfig.js`（站点配置）
- **语文**：编辑 `js/data.js`（所有知识点数据）
- **基础英语 850**：编辑 `js/wordData.js`（词汇数据）

详见各站点 README 的自定义章节。
