## 项目简介

本项目是一个**纯静态、无依赖、离线可用**的语文知识学习网站，专为人教版一年级下册教材配套设计。

- 适合对象：一年级小学生及家长
- 学习目标：辅助记忆背诵、词语积累、句式练习
- 使用方式：直接双击 `index.html` 打开，无需服务器、无需联网

## 快速开始

**方式一：直接打开（推荐）**

```bash
# 双击打开首页即可，无需任何配置
open index.html   # macOS
start index.html  # Windows
```

**方式二：本地服务器（如遇跨域问题）**

```bash
# 使用 Python 快速启动本地服务器
python -m http.server 8080
# 访问 http://localhost:8080
```

## 技术说明

- **技术栈**：纯 HTML5 + CSS3 + 原生 JavaScript，零外部依赖
- **字体**：系统字体回退（无需额外安装）
- **兼容性**：支持主流浏览器（Chrome / Edge / Firefox / Safari）
- **离线支持**：全部内联，断网可用
- **进度存储**：使用 `localStorage` 在本地保存已掌握状态，刷新页面不丢失

### 进度追踪机制

每个知识点卡片均有「已掌握 ✓」标记按钮。状态通过 `storage.js` 写入浏览器本地存储，跨页面共享，关闭重开后仍然保留。清除浏览器缓存会重置进度。
